Step 1: Define Models for Publisher and Author

Add models for the Publisher and Author with appropriate fields and relationships in the models.py file of the app above the book model.

models.py

  • class Author(models.Model):
  •     name = models.CharField(max_length=100)
  •     country = models.CharField(max_length=100)
  •     bio = models.TextField()
  •     website = models.URLField(blank=True)
  •     def __str__(self):
  •         return self.name
  • class Publisher(models.Model):
  •     name = models.CharField(max_length=150)
  •     established_date = models.DateField()
  •     country = models.CharField(max_length=100)
  •     website = models.URLField(blank=True)
  •     def __str__(self):
  •         return self.name

Step 2: Add publisher and author to the book model

Add publisher and authors fields to the book model

models.py

  • # Many-to-Many relationship with Author
  • authors = models.ManyToManyField(Author)
  • # One-to-Many relationship with Publisher
  • publisher = models.ForeignKey(Publisher, on_delete=models.CASCADE)

Step 6: Create Serializer for Author and Publisher Models

In my_app/serializers.py file, add serializer classes for Author and Publisher Models

serializers.py

  • from .models import Author,Book,Publisher
  • class AuthorSerializer(serializers.ModelSerializer):
  •     class Meta:
  •         model = Author
  •         fields = '__all__'
  • class PublisherSerializer(serializers.ModelSerializer):
  •     class Meta:
  •         model = Publisher
  •         fields = '__all__'
