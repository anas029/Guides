Step 1: Install dj-rest-auth, django-allauth

Install the required Python packages for your django authentication.

Windows

  • pip install dj-rest-auth django-allauth

Mac

  • pip3 install dj-rest-auth django-allauth
2
Step 2: Start a New Django App

In the terminal

Windows

  • python manage.py startapp token_auth

Mac

  • python3 manage.py startapp token_auth
3
Step 3: Configure settings.py

Add to INSTALLED_APPS

settings.py

  • 'rest_framework.authtoken',
  • 'allauth',
  • 'allauth.account',
  • 'dj_rest_auth',
  • 'dj_rest_auth.registration',
  • 'token_auth',

Add at the end of the file

settings.py

  • REST_FRAMEWORK = {
  • "DEFAULT_AUTHENTICATION_CLASSES": [
  • "rest_framework.authentication.TokenAuthentication",
  • ]
  • }
  • SITE_ID = 1
4
Step 4: Create urls file in token_auth

Inside token_app folder create a file named "urls" and add the below commands:

token_auth/urls.py

  • from django.urls import path
  • from dj_rest_auth.registration.views import RegisterView
  • from dj_rest_auth.views import LoginView, LogoutView
  • urlpatterns = [
  • path('register/', RegisterView.as_view(), name="rest_register"),
  • path('login/', LoginView.as_view(), name='rest_login'),
  • path('logout/', LogoutView.as_view(), name='rest_logout'),
  • ]
5
Step 5: Include token_app/urls in project/urls

Open my_project/urls.py and add the command inside urlpatterns

my_project/urls.py

  • path('auth/', include('token_auth.urls')),
6
Step 6: Add Authentication to Views

Open my_app/views.py and add this command inside class views where applicable

An example for Retrieve view:

my_app/views.py

  • from rest_framework.permissions import IsAuthenticated
  • class BookListRetrieve(RetrieveAPIView):
  •     permission_classes = [IsAuthenticated]
  •     queryset = Book.objects.all()
  •     serializer_class = BookSerializer
